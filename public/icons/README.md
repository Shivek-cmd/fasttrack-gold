# Favicon Files

Generate all favicon files from your logo at: https://realfavicongenerator.net

Upload a 512×512 SVG or PNG of your logo, download the package, place these files here:

- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png          (180×180)
- android-chrome-192x192.png    (192×192)
- android-chrome-512x512.png    (512×512)
- site.webmanifest              (already created ✓)

Also add your OG image to: /public/og/og-default.jpg (1200×630px)
